# Mobile Application Design and Development

## Project setup

### Step 1: Install package

```
npm install
```

### Step 2: Run serve
```
ionic serve
```

## Project build

### Option 1: Build Android
```
ionic capacitor add android
```

### Option 2: Build IOS
```
ionic capacitor add ios
```
